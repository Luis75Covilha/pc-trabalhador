# PC Trabalhador

Sistema de reaproveitamento de computadores antigos para execução automática de tarefas repetitivas em ambiente Windows.

## Resumo

O **PC Trabalhador** reutiliza computadores antigos, configurando-os para executarem tarefas repetitivas de forma automatizada, sem intervenção humana. A solução permite poupar tempo, reduzir erros humanos e libertar pessoas de tarefas com horário fixo, sem despesas adicionais relevantes.

O primeiro PC Trabalhador nasceu em Outubro de 2018. Actualmente, há três máquinas em operação, duas em Maputo, Moçambique, e uma na Praia, Cabo Verde.

## O que faz

- Grava automaticamente emissões de rádio e televisão.
- Guarda ficheiros com nomes padronizados.
- Coloca ficheiros em pastas sincronizadas na cloud.
- Recolhe páginas online e capturas de ecrã.
- Gere ficheiros antigos para evitar falta de espaço.
- Reinicia automaticamente o computador para reduzir falhas acumuladas.
- Executa tarefas agendadas sem intervenção humana.

## Porquê computadores antigos?

A ideia central é dar nova utilidade a computadores que já não servem bem para uso manual diário, mas continuam capazes de executar tarefas simples, repetitivas e previsíveis. Um computador lento pode ser pouco prático para uma pessoa, mas continua útil se trabalhar sozinho, de tampa fechada, em horários definidos.

## Software utilizado

A solução foi construída com software gratuito ou de baixo custo, com destaque para o **RoboTask Lite**, usado para automatizar tarefas em Windows sem exigir programação.

Outras ferramentas podem ser usadas conforme o caso, incluindo:

- programas de gravação de áudio;
- sincronização com Google Drive ou outro serviço cloud;
- scripts em Node.js ou outras linguagens;
- ferramentas de captura de ecrã;
- acesso remoto para manutenção.
